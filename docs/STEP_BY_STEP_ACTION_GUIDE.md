### Phase 2: Copy and Rename Documents (90-120 minutes)

For each of the 18 ready documents, follow these steps:

#### Template Steps (Copy these for each document)

**Example: Admin Threads**

1. **Open the source document**
   - Go to: https://docs.google.com/document/d/13tTTaXo9z18dSAUFo1cxJLFY9AihA_G_cC0NHLYLzuU/edit
   - Or search Google Drive for: `_quick-threads-manage Requirements`

2. **Make a copy**
   - Click: File → Make a copy
   - A new document will be created with "(Copy)" appended

3. **Rename the copy**
   - Click the document title at the top
   - Change to: `ADMIN_THREADS_REQUIREMENTS`
   - Press Enter

4. **Move to target folder**
   - Click the folder icon next to the title
   - Navigate to: `Split Lease/docs/Pending/`
   - Click "Move here"

5. **Verify the move**
   - Open `Split Lease/docs/Pending/` folder
   - Confirm `ADMIN_THREADS_REQUIREMENTS` is present

6. **Keep original document**
   - Do NOT delete the original document
   - It remains in its original location as a backup
