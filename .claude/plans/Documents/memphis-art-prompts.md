## USAGE NOTES

### For AI Image Generators (Midjourney, DALL-E, Stable Diffusion)
- Use prompts exactly as written
- Add aspect ratio if needed: `--ar 16:9` (horizontal), `--ar 9:16` (vertical), `--ar 1:1` (square)
- For icons: Add `--ar 1:1` for square format
- For patterns: Add `--tile` or `seamless` keyword if supported

### Color Variations
- All prompts use the original color scheme:
  - Primary: `#4B47CE` for main elements
  - Light: `#DFDFF6` for subtle/background elements

### File Naming Convention
```
{category}-{variant}-{color}-{size}.{ext}
Examples:
- empty-no-listings-primary-512.svg
- success-booking-confirmed-teal-1024.svg
- pattern-squiggle-wave-purple-256.svg
```
