What Fees Do I Have to Pay Before Signing the Lease?

You can expect to pay an application fee and deposit before you sign a lease.

Before signing a lease, there are several fees you should anticipate paying. Understanding these costs upfront helps you budget appropriately for your move.

Common Fees Before Signing

1. Application Fee

Typically $30-$75, this non-refundable fee covers:

  • Credit check
  • Background check
  • Previous landlord verification

2. Security Deposit

Usually one month's rent, this refundable deposit protects the landlord against:

  • Unpaid rent
  • Property damage beyond normal wear and tear
  • Cleaning costs if you leave the unit dirty

3. First Month's Rent

Due at lease signing, this is your actual rent for the first month of tenancy.

4. Last Month's Rent (Sometimes)

Some landlords require the last month's rent upfront. This is in addition to the security deposit.

5. Pet Deposit/Fee

If you have pets, expect $200-$500 in additional fees.

6. Broker's Fee (NYC/Boston)

In NYC and Boston, expect 12-15% of annual rent if using a broker.

Example Total Cost

For a $2,000/month apartment:

  • Application fee: $50
  • Security deposit: $2,000
  • First month: $2,000
  • Last month (if required): $2,000
  • Total: $6,050+

Which Fees Are Refundable?

  • Refundable: Security deposit, last month's rent, pet deposit (sometimes)
  • Non-refundable: Application fees, broker fees, non-refundable pet fees

State Limits on Deposits

Some states cap how much landlords can charge for deposits. Check your state's tenant laws.
