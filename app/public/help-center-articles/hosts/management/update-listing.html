Can I update my listing after it's published?

Yes! You can update your listing anytime - photos, description, pricing, availability, house rules, and more. Keep your listing current to maximize bookings.

Full Control

Update any aspect of your listing whenever needed. Changes take effect immediately for new bookings.

What You Can Update

1. Photos

  • Add new photos
  • Delete old photos
  • Reorder photos (first photo is cover image)
  • Update captions
  • Show seasonal changes
  • Tip: Fresh photos every 6 months keep listing appealing

2. Title and Description

  • Change listing title
  • Update description text
  • Highlight new features or amenities
  • Seasonal updates ("Winter: heated bedroom!")
  • Improve SEO and appeal

3. Pricing

  • Change base nightly/monthly rate
  • Adjust cleaning fees
  • Update damage deposit amount
  • Add or remove seasonal pricing
  • Weekend vs. weekday rates
  • Note: Only affects new bookings, not existing ones

4. Availability and Calendar

  • Block dates
  • Open new dates
  • Change available days of week
  • Update minimum stay requirements
  • Modify advance notice period

5. Amenities

  • Add new amenities (installed AC, added parking)
  • Remove amenities (WiFi down temporarily)
  • Update amenity descriptions

6. House Rules

  • Change policies (smoking, pets, guests)
  • Update quiet hours
  • Modify check-in/check-out times
  • Add seasonal rules
  • Important: Existing guests bound by original rules unless they agree to changes

7. Property Details

  • Update square footage
  • Change bedroom/bathroom count
  • Modify property type
  • Update neighborhood description

How to Update Your Listing

Step-by-Step

  1. Log into Split Lease account
  2. Go to Dashboard ‚Üí "Listings"
  3. Select listing to update
  4. Click "Edit Listing"
  5. Make your changes
  6. Review changes
  7. Click "Save" or "Publish Changes"
  8. Changes live immediately

Preview Before Publishing

  • See how changes look
  • Test from guest perspective
  • Check mobile and desktop views
  • Ensure photos display correctly

When Changes Take Effect

Immediate Changes

  • Photos
  • Description text
  • Amenities list
  • New availability blocks
  • Pricing (for new bookings)

Existing Bookings

  • Not affected: Guests keep original terms they booked under
  • Price locked: Can't change price for existing bookings
  • House rules: Original rules apply unless guest agrees to changes
  • Exception: Adding amenities benefits existing guests too

Update Strategies

Regular Refresh (Every 3-6 Months)

  • New photos showing seasonal changes
  • Update description with guest feedback themes
  • Add new amenities or improvements
  • Refresh keeps listing algorithm-friendly

Performance-Based Updates

If bookings are slow:

  • Review and improve photos
  • Rewrite description to be more compelling
  • Lower price slightly
  • Reduce minimum stay requirement
  • Add amenities that guests are asking for

If bookings are strong:

  • Consider raising price
  • Add premium amenities
  • Increase minimum stay for better guests

Seasonal Updates

  • Winter: Highlight heating, proximity to indoor attractions
  • Summer: AC, outdoor space, near beach/parks
  • Photos: Show property in current season

Optimizing Your Listing

Photos

Best practices:

  • 20-30 high-quality photos
  • Cover photo is your best shot
  • Show every room guest will access
  • Natural light, clean space
  • Mix wide shots and detail shots
  • Include neighborhood/building exterior

Title

Compelling titles:

  • "Sunny 2BR in Williamsburg - Walk to L Train"
  • "Modern Studio with Parking - Financial District"
  • "Cozy Room in Quiet Home - Perfect for Commuters"
  • Include: Style, size, neighborhood, key feature

Description

Structure:

  • Opening: Who is this perfect for?
  • The Space: What guest gets (room, bathroom, storage)
  • The Building: Amenities, elevator, laundry
  • The Neighborhood: Transit, food, attractions
  • House Rules: Quick summary
  • Length: 150-300 words

Pricing

Competitive pricing:

  • Research similar listings in your area
  • Price 5-10% below if new listing
  • Increase as you get positive reviews
  • Premium listings can charge 20-30% more

A/B Testing

Test Different Approaches

  • Change title, track inquiries for 2 weeks
  • Swap cover photo, measure click-through
  • Adjust price, monitor booking rate
  • Keep what works, change what doesn't

Metrics to Track

  • Views: How many people see your listing
  • Click-through rate: Views ‚Üí detail page
  • Inquiry rate: Views ‚Üí inquiries
  • Booking conversion: Inquiries ‚Üí bookings

Responding to Feedback

Guest Suggestions

  • Read all guest reviews
  • Note common requests
  • Update listing to address
  • Add amenities guests want

Example: Multiple guests ask about desk space ‚Üí Add desk, update photos, mention in description

Common Complaints

  • If guests say "not as pictured" ‚Üí Update photos to be accurate
  • If "too expensive" ‚Üí Review pricing or add value
  • If "not clean" ‚Üí Deep clean, update standards, show in photos

Things You Cannot Change

Core Listing Elements

  • Property address: Fixed (create new listing if moving)
  • Property type: Can't change from private room to entire place (major change)
  • Existing booking terms: Locked for confirmed guests

Workarounds

  • Major property type change ‚Üí Create new listing
  • Different address ‚Üí New listing required
  • Existing guest terms ‚Üí Communicate directly, mutual agreement needed

Bulk Updates for Multiple Properties

If You Have Multiple Listings

  • Update each separately (no bulk edit currently)
  • Use same description template, customize per property
  • Consistent pricing strategy across portfolio
  • Sync calendars if properties share availability

Best Practices

Keep Listing Accurate

  • Update immediately if something breaks
  • Add new amenities as you add them
  • Remove amenities if unavailable
  • Accurate listings = happy guests = good reviews

Regular Maintenance

  • Monthly: Review and update availability
  • Quarterly: Refresh photos, update description
  • Annually: Complete listing audit and overhaul

Announce Major Updates

  • Message existing/upcoming guests about big changes
  • Positive updates (new AC!) delight guests
  • Negative updates (construction nearby) set expectations

Common Update Scenarios

You Added Amenities

  1. Take new photos showing amenity
  2. Update amenities list
  3. Mention in description
  4. Consider raising price if significant upgrade

Neighborhood Changed

  • New subway line opened ‚Üí Update transit info
  • Great restaurant nearby ‚Üí Mention in description
  • Construction starting ‚Üí Warn in listing

Pricing Strategy Shift

  • Slow season ‚Üí Lower price, flexible rules
  • Peak season ‚Üí Raise price, longer minimum stay
  • Update calendar and pricing together

Summary

Listing Updates

  • Update anytime: photos, description, pricing, rules
  • Changes immediate for new bookings
  • Existing bookings keep original terms
  • Refresh listing every 3-6 months
  • Respond to guest feedback with updates
  • Track metrics to measure improvement
  • Keep listing accurate and current

Your listing is a living document. Keep it fresh, accurate, and optimized to attract the best guests at the best rates.
