## Common UI Patterns

### Button Variants

```css
.btn-primary {
  background: var(--gradient-purple-button);
  color: white;
  border: none;
}

.btn-secondary {
  background: var(--bg-light-gray);
  color: var(--secondary-purple);
  border: none;
}

.btn-outline {
  background: transparent;
  color: var(--secondary-purple);
  border: 1px solid var(--border-color);
}

.btn-ghost {
  background: transparent;
  color: var(--secondary-purple);
  border: none;
}
```

### Card Pattern

```css
.card {
  background: var(--bg-white);
  border: 1px solid var(--border-color);
  border-radius: var(--rounded-lg);
  padding: var(--spacing-lg);
  box-shadow: var(--shadow-sm);
}
```

### Modal Pattern

```css
.modal-backdrop {
  background: rgba(0, 0, 0, 0.5);
  position: fixed;
  inset: 0;
  z-index: var(--z-modal);
}

.modal-content {
  background: var(--bg-white);
  border-radius: var(--rounded-xl);
  max-width: var(--max-width-modal);
  box-shadow: var(--shadow-xl);
}
```

### Day Selection Pattern

The app uses JavaScript's 0-based day indexing:

| Index | Day |
|-------|-----|
| 0 | Sunday |
| 1 | Monday |
| 2 | Tuesday |
| 3 | Wednesday |
| 4 | Thursday |
| 5 | Friday |
| 6 | Saturday |

## Protocol Design System

For specific UI components, the app uses a stricter "Protocol" design system:

| Token | Value | Usage |
|-------|-------|-------|
| `--protocol-primary` | `#31135D` | Primary brand |
| `--protocol-action` | `#5B5FCF` | Action buttons |
| `--protocol-danger` | `#DC3545` | Destructive actions |
| `--protocol-space-sm/md/lg` | `8/16/24px` | Strict spacing (8px grid) |
| `--protocol-radius-button` | `100px` | Pill buttons |
| `--protocol-radius-modal` | `16px` | Modal corners |

**Important:** Protocol colors exclude green and yellow - using monochromatic purple only.

## Usage in Components

### CSS Variables

```css
.my-component {
  color: var(--text-dark);
  background: var(--bg-white);
  padding: var(--spacing-lg);
  border-radius: var(--rounded-md);
  box-shadow: var(--shadow-sm);
  transition: all var(--transition-base) ease;
}

.my-component:hover {
  box-shadow: var(--shadow-md);
}
```

### Responsive Design

```css
.my-component {
  padding: var(--spacing-md);
}

@media (min-width: 768px) {
  .my-component {
    padding: var(--spacing-xl);
  }
}
```
